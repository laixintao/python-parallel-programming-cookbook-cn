使用 ``queue`` 进行线程通信
============================

前面我们已经讨论到，当线程之间如果要共享资源或数据的时候，可能变的非常复杂。如你所见，Python的threading模块提供了很多同步原语，包括信号量，条件变量，事件和锁。如果可以使用这些原语的话，应该优先考虑使用这些，而不是使用queue（队列）模块。队列操作起来更容易，也使多线程编程更安全，因为队列可以将资源的使用通过单线程进行完全控制，并且可以使用“清理着”等可读性更高的设计模式。

Queue常用的方法有以下四个：

- ``put()``: 往queue中放一个item
- ``get()``: 从queue删除
